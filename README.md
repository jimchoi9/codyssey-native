# codyssey-native

## B1-1 — 비즈니스 메일 자동화 AI 비서

`B1-1` 폴더는 "비즈니스 커뮤니케이션 AI 비서" 과제의 산출물 모음입니다. 각 파일의 내용은 다음과 같습니다.

### 📄 파일별 설명

| 파일 | 설명 |
| :--- | :--- |
| `system_design_document.md` | **[산출물 1] 시스템 설계 문서.** 해결하려는 업무 문제, 타겟 사용자, AI 페르소나, 시스템 프롬프트(안전장치·사실 처리 규칙·형식 준수), 사용자 입력 템플릿(유저 프롬프트), Few-shot 예시 등 AI 비서의 설계 전반을 정의합니다. |
| `model_comparison_report.md` | **[산출물 2] LLM 모델 비교·선정 보고서.** GPT-5.5 Instant·Claude Sonnet 4.6·Gemini 3.5 Flash 세 모델을 5개 축(형식 준수, 긴 문맥 유지, 속도/안정성, 비용, 사용자 편의성)으로 1~5점 채점하여 비교하고, 최종적으로 Claude Sonnet 4.6(22점)을 선정한 근거를 정리합니다. 재현성 메타데이터(모델명, 채널, 요금제, 설정값, 사용 날짜)도 함께 기록합니다. |
| `execution_log.md` | **[산출물 3] 실행 로그(대화 로그).** 외부 파트너사(B회사)에 미팅 요청 메일을 작성하는 시나리오로 10턴 이상 대화를 진행한 전문입니다. 필수 정보 누락 시 확인 질문을 던지는 **안전장치 검증**, 톤 변경·추가 정보 지연 제공에 대한 **문맥 유지 검증**, 불확실한 정보 요구 시 사실을 지어내지 않는 **환각 방어 검증**을 다룹니다. |

### 📁 logs 폴더

| 파일 | 설명 |
| :--- | :--- |
| `logs/model_compare_logs.md` | 모델 비교 보고서(`model_comparison_report.md`)의 근거가 된 **원본 응답 로그.** 동일한 메일 작성 프롬프트에 대해 GPT-5.5 Instant·Claude Sonnet 4.6·Gemini 3.5 Flash가 실제로 생성한 출력 전문을 모델별로 담고 있습니다. |
| `logs/[실행로그]비즈니스 커뮤니케이션 AI 비서 .pdf` | 비즈니스 커뮤니케이션 AI 비서의 **실행 로그 PDF 캡처본.** 대화 화면 등 실제 실행 증빙 자료입니다. |
